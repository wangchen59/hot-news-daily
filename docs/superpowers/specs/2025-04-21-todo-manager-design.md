# Todo Manager 设计文档

## 概述

网页版Todo管理器，支持任务分类、优先级、截止日期、筛选和拖拽排序。纯前端实现，数据存储于localStorage。

## 目标

- 现代简约风格，减少视觉干扰，专注效率
- 标准功能：分类、优先级、截止日期、筛选
- 混合排序：自动排序 + 手动拖拽覆盖

---

## 二、数据结构

### 任务对象

```javascript
{
  id: "uuid",           // 唯一标识
  title: "任务标题",     // 必填
  category: "work",     // 分类: work/personal/study/other
  priority: "medium",   // 优先级: high/medium/low
  dueDate: "2025-04-25",// 截止日期（可选）
  completed: false,     // 完成状态
  customOrder: null,    // 手动排序位置（拖拽后设置，null表示自动排序）
  createdAt: timestamp  // 创建时间戳
}
```

### 预设分类

| 值 | 显示名称 | 标签颜色 |
|---|---------|---------|
| work | 工作 | 蓝色 |
| personal | 个人 | 绿色 |
| study | 学习 | 紫色 |
| other | 其他 | 灰色 |

### 优先级

| 值 | 显示名称 | 标记颜色 |
|---|---------|---------|
| high | 高 | 红色 |
| medium | 中 | 黄色 |
| low | 低 | 灰色 |

## 三、交互设计

### 任务操作

| 操作 | 交互方式 |
|------|----------|
| 新建任务 | 点击"新建"按钮 → 表单模态框（标题、分类、优先级、截止日期） |
| 编辑任务 | 点击卡片"编辑"图标 → 相同表单模态框 |
| 删除任务 | 点击"删除"图标 → 确认对话框 → 删除 |
| 标记完成 | 点击卡片左侧圆形checkbox → 切换状态（标题变灰+划线） |
| 拖拽排序 | 拖动卡片 → 释放后保存customOrder值 |

### 筛选逻辑

- **状态筛选**：全部 / 进行中 / 已完成
- **分类筛选**：全部 / 工作 / 个人 / 学习 / 其他
- **优先级筛选**：全部 / 高 / 中 / 低
- **搜索**：标题关键词模糊匹配

### 模态框表单字段

| 字段 | 类型 | 必填 |
|------|------|------|
| 标题 | 文本输入 | 是 |
| 分类 | 下拉选择 | 否（默认"其他"） |
| 优先级 | 下拉选择 | 否（默认"中"） |
| 截止日期 | 日期选择 | 否 |

---

## 四、排序规则（混合模式）

1. 如果任务有 `customOrder` 值 → 按 customOrder 升序排列
2. 否则 → 自动排序：
   - 优先级：高 → 中 → 低
   - 截止日期：早 → 晚（无截止日期排在最后）
   - 创建时间：新 → 旧

## 五、技术方案

### 前端架构

- **HTML**：语义化标签，响应式布局
- **CSS**：现代简约风格，Flexbox/Grid布局，媒体查询适配移动端
- **JavaScript**：原生JS，无框架依赖

### 数据持久化

- `localStorage` 存储任务列表JSON
- 键名：`todoManager_tasks`

### 拖拽实现

- 原生 `dragstart` / `dragover` / `drop` 事件API
- 拖拽完成后计算新位置并更新所有受影响任务的 `customOrder`

### 文件结构

```
output/
├── todo.html          # 主页面
├── css/
│   └── todo.css       # 样式文件
└── js/
    └── todo.js        # 逻辑脚本
```

## 八、响应式适配

| 断点 | 布局调整 |
|------|----------|
| > 768px | 篮选栏横向排列，卡片正常宽度 |
| ≤ 768px | 篮选栏改为下拉菜单或折叠，卡片全宽 |
| ≤ 480px | 模态框全屏显示，按钮文字简化 |

## 九、功能边界

**包含：**
- 任务CRUD操作
- 分类、优先级、截止日期
- 状态/分类/优先级筛选 + 搜索
- 拖拽排序
- localStorage持久化
- 响应式布局

**不包含：**
- 用户登录/云端同步
- 子任务/任务依赖
- 任务提醒/通知
- 任务统计/图表
- 自定义分类添加
